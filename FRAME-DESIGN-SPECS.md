## General Design Guidelines

### ✅ DO:
1. **Design di SAFE AREAS saja** (margins, gaps between slots)
2. **Transparant PNG** dengan alpha channel
3. **Photo slots HARUS transparant** 100% - jangan design apa-apa di area foto
4. **Resolution 300 DPI** untuk print quality
5. **Warna konsisten** dengan ikutpose branding (pink theme)
6. **Readable text** di top/bottom margins untuk branding
7. **Decorative elements** di side margins (hearts, stars, icons)
8. **Border/frame** bisa di sekitar photo slots dengan padding

### ❌ DON'T:
1. **Jangan design di area photo slots** (harus transparant)
2. **Jangan text terlalu kecil** (<12px at 300dpi akan susah dibaca)
3. **Jangan overflow** keluar canvas boundaries
4. **Jangan gradient opacity** pada photo slots (harus 100% transparant)
5. **Jangan file size terlalu besar** (max 5MB per file)

### Branding Elements to Include:
- **Logo/text:** "ikutpose photobooth" atau "ikutpose.id"
- **Social media:** Instagram (@ikut.pose), TikTok (@ikut.pose)
- **Decorative:** Hearts, stars, camera icons (match dengan existing pink theme)
- **Style:** Kawaii/cute, modern, feminine, pink-based palette

### File Naming Convention:
- Classic Strip 4: `strip_4_1200x1800.png`
- Grid 2x2: `grid_2x2_1800x1800.png`
- Single Portrait: `single_1200x1600.png`
- Wide Strip 3: `wide_strip_3_2400x1200.png`
- Strip 3 Compact: `strip_3_1200x1500.png`
- Strip 2 Big: `strip_2_1200x1600.png`

### Upload Instructions:
1. Upload ke `/var/www/ikutpose.id/frontend/public/overlays/`
2. Update database:
   ```sql
   UPDATE templates SET overlay_image = '/overlays/strip_4_1200x1800.png' WHERE layout = 'strip_4';
   ```
3. Test di https://ikutpose.id/booth
